## Prerequisites

Before starting any tutorial, make sure you have:

1. **ReasonDB running** - Follow the [Quickstart](/quickstart) to set up
2. **An LLM API key** - Anthropic, OpenAI, Gemini, or Cohere
3. **curl or an HTTP client** - For making API requests

## Tutorial Format

Each tutorial includes:

- **Goal**: What you'll build
- **Sample Data**: Pre-made documents to ingest
- **Step-by-Step Instructions**: Detailed walkthrough
- **Working Examples**: Copy-paste curl commands
- **Challenges**: Optional exercises to extend your learning
